# 📊 Nitesh Ecommerce Sales Dashboard (Power BI)

## 🚀 Project Overview
This project presents an interactive **Ecommerce Sales Dashboard** built using Power BI. It provides meaningful insights into sales performance, profit trends, customer behavior, and product category analysis.

The dashboard is designed to help businesses make **data-driven decisions** through clear and interactive visualizations.

## 🎯 Objectives
- Analyze overall sales and profit performance
- Track key business metrics (Sales, Quantity, Profit, AOV)
- Identify top-performing states and customers
- Understand category-wise and sub-category-wise trends
- Monitor monthly profit and loss patterns
- Analyze payment method preferences
